# API Reference

TypeDoc-generated documentation for the `easy-eval` package — useful when writing custom judges, schemas, or extending the framework.

## Top-level exports

The package exports:

- **`defineConfig`** — type-safe config builder
- **`vibecheck`**, **`exactMatch`**, **`fuzzyMatch`**, **`llmJudge`** — built-in judges
- **`fromZod`** — derive a diff schema from a Zod schema
- Core types: `EvalConfig`, `EvalDef`, `EvalContext`, `EvalMethod`, `EvalVerdict`, `JudgeInput`, `Golden`, `EvalRun`, `DiffResult`, `SectionConfig`, `Change`, and others.
